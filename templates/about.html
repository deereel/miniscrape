Company Information Scraping Tool

About MiniScrape

MiniScrape is a powerful web scraping tool designed to extract company information from various online sources. It automates the process of gathering data such as company names, addresses, and officer information from websites, LinkedIn, and other public records.

Key Features

  • Extract company information from websites
  • Support for LinkedIn profile scraping
  • Access to Companies House database
  • Multiple input methods: manual entry or file upload
  • Results export to CSV and Excel formats
  • Responsive web interface
  • Robust error handling and retry mechanisms

How It Works

1

Input URLs

Enter company websites or upload a file with multiple URLs

2

Start Scraping

Click the scrape button to begin data extraction

3

View Results

See extracted company information in real-time

4

Export Data

Download results as CSV or Excel files

Technology Stack

Flask Web framework for the interface
Python Core scraping engine
Beautiful Soup HTML parsing
Selenium Dynamic content scraping
Pandas Data handling and export
Requests HTTP communication

Usage Instructions

Basic Usage

1. Enter web addresses one per line or separated by commas in the text area 2. Or upload a file containing URLs (.txt or .xlsx format) 3. Click the "Scrape" button to start the process 4. View results in the table below 5. Download results as CSV or Excel files

File Formats

Supported file formats:

  • Text files (.txt): URLs separated by newlines or commas
  • Excel files (.xlsx): Single column with URLs in first worksheet

Output Formats

Results can be downloaded as:

  • CSV: Comma-separated values for spreadsheets
  • Excel: XLSX format with formatted cells
